# @algosail/sail

A small functional programming library for vanilla JavaScript. Provides curried utilities and algebraic data types — `Maybe`, `Either`, `Validation`, `Pair`, lenses, and more — across 20 modules with ~420 functions.

## Install

```sh
npm install @algosail/sail
```
